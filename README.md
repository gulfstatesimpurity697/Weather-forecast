## 🖥️ What This App Does

Weather-forecast is a weather app for Windows users who want quick, clear weather details without extra steps.

- 5-day forecast

It uses live weather data from OpenWeatherMap and updates based on the place you choose.

### The page stays blank

- Refresh the page
- Check your internet connection
- Open the app in another browser

### Weather data does not load

- Make sure the internet is working
- Check that location access is allowed
- Try searching a city name by hand

### The app does not open

- Make sure you extracted the ZIP file
- Look for the correct start file
- Try opening the file in Microsoft Edge or Chrome

### Search results are wrong

- Type the city name with correct spelling
- Add the country if needed, such as `Paris, FR`
- Wait a moment and try again
