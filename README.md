# 🌦️ Weather Forecast

A real-time weather application that fetches live weather data using OpenWeatherMap API.

## 🚀 Features
- 🌍 Location-based weather
- 🔍 Search by city
- 🌡️ Real-time temperature & conditions
- 📅 5-day forecast
- 📱 Fully responsive design
- 🌙 Dark mode (optional)
- ⚡ Fast and optimized UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## 🔑 Setup
1. Get API key from https://openweathermap.org/api
2. Enter API key in app (modal)
3. Start exploring weather 🌤️
